Latest new features & fixes to the NOW database

  • Added a FAQ page
  • Added “unworn” to mesowear categories
  • Added a list of metadata of the website, including the major contributors of the last few months
  • Added into the download file species that have no public locality (interesting because of traits)
  • Updated the NOW Advisoy Board listing
  • Fix: Trim all taxa before saving them in the database.
