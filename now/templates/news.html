News

Latest news and Circulars

  • The NOW www-pages have been updated to manage new HTML5 semantic elements like <header>, <footer>, <article>, and <section>.
  • November 7, 2019: A bit on how NOW works.[PDF]

  • November 6, 2019: Announcing NOW book.[PDF]

  • October 14, 2019: Indrė Žliobaitė replaces Mikael Fortelius as the General Coordinator of NOW. A new NOW body, the Emeritus Board has been set up.[PDF]

  • December 12, 2013: Lars van den Hoek Ostende as the Associate Coordinator. Steering Group for NOW has been set up.[PDF]

  • June 8, 2010: Initiating a concerted effort to come to grips with NOW synonyms.[PDF]

  • April 29, 2010: New Board listing on the NOW homepage.[PDF]

  • October 23, 2009: North American Advisory Board for NOW has been set up. New PHP-based interface launched. Sections for Australia as well as for South America.[PDF]

  • February 15, 2009: Data from the Tertiary Mammals of North America monographs into NOW. New suggestion for NOW - "New and Old Worlds".[PDF]

  • December 19, 2006: It’s ten years since the official launch of the NOW database as a public resource. [PDF]

  • June 3, 2002: NOW is available in the Biodiversity U.K. home page. NOW data are being used in analyses that are beginning to catch the attention of the evolutionary biology community. [PDF]

  • May 14, 1998: Next year it will probably be possible to access a recent version of the public set of NOW data with a common web browser (such as Netscape).[PDF]

  • October 15, 1997: Creation of an Advisory Board for NOW. A few words about the "data policy".[PDF]

  • September 15, 1995: The Evolutionary History of Western Eurasia: A Synthesis of Regional Faunas, Their Geochronology and Paleoenvironments, 15-5 m. y. a. -- The Schloss Reisensburg ´92 Database.[PDF]

New features & fixes to the NOW database

  • Data: Body mass data has been added to 1258 species
  • Data: South American data added from Croft, D.A., 2016. Horned armadillos and rafting monkeys: the fascinating fossil mammals of South America.
  • Added new taxonomical fields: Subclass or Superorder, Suborder or Superfamily
  • Fix: coordinate services in the data entry
  • Data: New MN units (Hilgen, 2012) are added to the time units
  • Data: 'Findout' taxa updated as 'indet.'
  • Added an error reporting button (contact) for Locality, Species and Reference
  • Added a FAQ page
  • Added “unworn” to mesowear categories
  • Added a list of metadata of the website, including the major contributors of the last few months
  • Added into the download file species that have no public locality (interesting because of traits)
  • Updated the NOW Advisoy Board listing
  • Fix: Trim all taxa before saving them in the database.
